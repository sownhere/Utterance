# ``Utterance``

The Ultimate Speech-AI Pipeline for iOS & macOS.

## Overview

Utterance is a powerful, modular SDK designed to simplify building speech-enabled applications. From low-latency audio recording to real-time transcription and translation, Utterance provides a unified API surface that is both easy to use and highly extensible.

### Feature Highlights

- **Audio Recording**: Robust audio engine with ring-buffer architecture.
- **Transcription**: Real-time Speech-to-Text.
- **Translation**: On-device and API-based translation.
- **Waveform Visualization**: Beautiful, data-driven UI components.
